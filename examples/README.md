# How to run/build the examples

```bash
# Clone repo
git clone https://github.com/aliuq/vite-ssr-vue3.git
cd vite-ssr-vue3

# Install all project dependencies and build project
pnpm install
pnpm b

# Basic example
pnpm basic:dev
pnpm basic:build
pnpm basic:serve
pnpm basic:serve:bin
pnpm basic:serve:custom

pnpm basic:build:noexternal
pnpm basic:preview

# Vitesse lite and naive-ui
pnpm vl-naive:dev
pnpm vl-naive:build
pnpm vl-naive:serve
pnpm vl-naive:serve:bin
pnpm vl-naive:serve:custom

pnpm vl-naive:build:noexternal
pnpm vl-naive:preview
```
